# <a name="powerapps-component-framework-api-reference"></a>Referencia de la API del marco de componentes de PowerApps

Esta sección contiene documentación de referencia para crear componentes de código mediante el marco de componentes de PowerApps.

> [!IMPORTANT]
> En la pestaña **disponible para** se muestran las API que son compatibles con las aplicaciones controladas por modelos y las aplicaciones de lienzo (versión preliminar experimental). Se recomienda comprobar la sección **disponible** para cada uno de los métodos de la API, ya sea compatible o no. Por ejemplo, la API de **cliente** es compatible con aplicaciones controladas por modelos y aplicaciones de lienzo, pero el método **isOffline** de la API de **cliente** solo admite aplicaciones controladas por modelos.
